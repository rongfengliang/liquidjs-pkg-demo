# liquidjs with pkg demo

use typescript for code writing && pkg for  package binary file

## how to running

* install deps

```code
yarn or npm i
```

* live code compile

```code
yarn l
```

* local running

> may be run `yarn l`first

```code
yarn app  
```

* package binary file

```code
yarn p
```
